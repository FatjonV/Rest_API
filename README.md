Overview
This project is a backend application that combines GraphQL and WebSocket communication (via Socket.IO) to provide both flexible data querying and real-time updates.

It was developed as a step forward from traditional REST APIs, with the goal of learning more advanced backend concepts such as real-time communication, efficient data fetching, and event-driven architecture.

Purpose of the Project:
Move beyond REST API limitations
Learn GraphQL query structure and schema design
Implement real-time features using WebSockets
Understand event-driven backend systems

Features:
GraphQL API for flexible data querying
Real-time updates using Socket.IO
User authentication (JWT-based)
Create and manage resources 
Live updates without page refresh
Modular backend structure

Technologies Used:
Backend: Node.js, Express.js
API: GraphQL, REST-API
Real-time Communication: Socket.IO
Database: MongoDB 
Authentication: JSON Web Tokens (JWT)
